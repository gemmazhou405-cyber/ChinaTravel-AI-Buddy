# ChinaEase Buddy Design System — Warm Editorial

## Palette
- Background: #FAF8F5 (warm off-white)
- Surface: #FFFFFF
- Ink primary: #111418
- Ink secondary: #5B6470
- Ink tertiary: #8B929B
- Accent (jade): #0F5257
- Accent soft (jade wash): #E8F0EE (subtle backgrounds only)
- Border: #E8E4DC (hairline, warm)
- No other colors. No gradient hero.

## Typography
- Display (H1 only): Fraunces, weight 400, opsz 144, tracking -0.02em, line-height 1.05. Self-host / subset the font, do not hotlink Google Fonts CSS at runtime.
- UI + body: Inter, weights 400/500/600 only
- Scale desktop: 64/48/32/24/20/16/14
- Scale mobile: 40/32/24/20/18/16/14
- Body: 17px desktop, 16px mobile, line-height 1.55

## Spacing
- 4px base unit
- Section vertical padding: 128px desktop, 80px mobile
- Container max: 1140px, gutter 32/24
- Card padding: 32px desktop, 24px mobile

## Radius + Elevation
- Buttons: 8px / Cards: 16px / Modals: 20px
- One shadow token only: 0 1px 2px rgba(17,20,24,0.04), 0 8px 24px rgba(17,20,24,0.06)
- Everything else borderless or hairline border

## Motion
- 180ms ease-out hovers, 240ms ease-out modal entry
- No auto-playing hero animations
